In June this year, I get started with hadoop and always wonder its meaning to me. After reading a book named Data Intensive Text Processing with Mapreduce, I gradually know what hadoop can do and what can not.

Hadoop suits batch processing tasks well, but no interaction among sub-tasks limits its usage. For example, top down or bottom up based word clustering can not be handled by hadoop. Great efforts have been made to push hadoop into industrial usage. As we see, mahout is an open source project founded by Apache, which contains implementation of several machine learning algorithms, such as k-means, naive bayes, recommendation system, etc. If you are interested, download from here.

Can hadoop train a BP neural network or not? This question troubles me a lot. You probably see stocastic gradient descent, batch gradient descent and gradient descent are often the ways to optimize weight matrixes in neural network trainning process.
